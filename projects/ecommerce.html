Overview

Project Overview

A fully-featured e-commerce platform built from the ground up with Laravel. The system covers the entire shopping lifecycle — from product discovery and cart management to checkout, order tracking, and admin analytics — all with a smooth, page-refresh-free experience.

Livewire powers the admin dashboard for real-time order management, while AJAX handles storefront interactions like cart updates and wishlist toggles without any page reloads.

Features

User Interface
  • Fully Responsive — Mobile-first design that works perfectly on all screen sizes
  • Real-time Order Tracking — Customers can monitor their order status live without refreshing
  • Discount Coupons — Flexible coupon system to boost sales and customer engagement
  • Wishlist — Save favorite products for later with instant AJAX toggle
  • Email Verification — Secure account activation via email confirmation
  • Password Reset — Self-service password recovery flow via email
Admin Dashboard
  • Real-time Order Management — Livewire-powered dashboard with live order updates and status control
  • Product & Category Control — Full CRUD for products, categories, and inventory management
  • User Management — View, activate, or deactivate customer accounts
  • Coupon Management — Create and manage dynamic discount codes with expiry and usage limits
  • Sales Reports & Analytics — Real-time charts and revenue breakdowns
Performance Enhancements
  • Livewire — Real-time UI updates for cart, orders, and admin panels without full page reloads
  • AJAX — Smooth and fast storefront interactions including wishlist, cart, and filters
  • Swiper.js — Interactive product sliders and carousels for a modern shopping feel
  • ScrollReveal.js — Smooth entrance animations on product listings and sections

Architecture

Technical Decisions
Livewire over full SPA — Chose Livewire to keep the Laravel ecosystem intact while delivering real-time UI without a separate JS framework or API layer.
Session-based cart — Cart state is maintained server-side in the session, ensuring consistency across tabs and preventing client-side manipulation.
Middleware-based auth — Route groups protected with custom middleware, separating guest, user, and admin access cleanly.
Queued email jobs — Verification and password reset emails are dispatched via Laravel queues to avoid blocking the request lifecycle.
