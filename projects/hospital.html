Overview

Project Overview

A comprehensive Hospital Management System designed to streamline medical facility operations end-to-end. The system connects every stakeholder — admin, doctors, patients, lab staff, and radiology personnel — through dedicated dashboards, each tailored to their specific workflow.

Built with a clean Repository Pattern architecture for maintainability and scale, and fully bilingual (Arabic & English) via mcamara/laravel-localization and astrotomic/laravel-translatable.

Features

Medical Management
  • Departments & Services — Manage hospital sections with single & group medical services
  • Appointments — Online booking system with real-time availability checking
  • Diagnostics — Record and track patient diagnoses and full medical history
  • Radiology Module — Specialized workflow for ray/imaging requests and results delivery
  • Laboratory Module — Full lab test lifecycle from request to result delivery
Financial System
  • Invoicing — Generate single and group service invoices automatically
  • Patient Accounts — Full account ledger per patient with transaction history
  • Receipts & Payments — Track all transactions with detailed records
  • Fund Accounts — Manage hospital financial accounts and balances
Integrations & Operations
  • Insurance Companies — Patient coverage verification and billing integration
  • Ambulance Fleet — Track and dispatch ambulances in real time
  • Chat System — Internal messaging between staff and patients
  • Real-time Notifications — Pusher-powered live alerts + Email via SMTP

Architecture

Backend Architecture
Repository Pattern — Business logic is fully decoupled from the framework core, ensuring clean, testable, and maintainable code.
Modular Design — Separated logic for Appointments, Billing, Laboratories, Radiology, and Patient Records.
Query Optimization — Extensive use of eager loading and indexing to handle large medical datasets without performance degradation.
Bilingual Support — Full Arabic / English localization using mcamara/laravel-localization and astrotomic/laravel-translatable.
